# Twitter/X 動画・画像ダウンローダー

Twitter/X の動画・画像にマウスをかざして右クリックするだけでダウンロードできる Chrome 拡張機能。

## 使い方

1. 動画または画像にマウスをかざす → 右上に **「⬇ ダウンロード」** ボタンが表示される
2. そのままクリック、または**右クリック → ダウンロード** を選択

ダウンロードフォルダに直接保存されます。動画は複数画質がある場合、自動で最高画質を選択します。

### インストール

1. このリポジトリを ZIP でダウンロードして解凍
2. Chrome で `chrome://extensions` を開く
3. 右上の **デベロッパーモード** をオン
4. **「パッケージ化されていない拡張機能を読み込む」** → 解凍したフォルダを選択

## 対応コンテンツ

| コンテンツ | 対応 |
|---|---|
| ツイートの動画 | ✅（最高画質を自動選択）|
| ツイートの画像 | ✅（オリジナル解像度）|
| GIF | ✅（MP4 として保存）|

## 仕組み

- **動画**: ブラウザが読み込んだ `video.twimg.com/*.mp4` を検出し、最高解像度のものをダウンロード
- **画像**: `pbs.twimg.com/media/` の URL を `name=orig`（オリジナル解像度）に変換してダウンロード
- Twitter の独自右クリックメニューより先にイベントを捕捉するため、通常の右クリックと同じ操作感で使える

## 動作環境

- Chrome 102 以上
- Windows / Mac 対応
